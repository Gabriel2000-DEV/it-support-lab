# Guia de Troubleshooting

##  Sem acesso à internet
- Verificar cabo/conexão Wi-Fi
- ipconfig /all
- Verificar IP (APIPA 169.254.x.x)
- ping 8.8.8.8
- nslookup google.com

---

##  Máquina lenta
- Verificar uso de CPU/RAM
- Task Manager / top
- Finalizar processos pesados
- Verificar inicialização

---

##  Problemas de login
- Verificar usuário
- Resetar senha
- Conferir bloqueio de conta

---

## Problema de DNS
- nslookup google.com
- Alterar DNS para:
  - 8.8.8.8
  - 8.8.4.4
