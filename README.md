# IT Support Lab 

Projeto prático simulando um ambiente real de Suporte Técnico em TI (Help Desk / Infraestrutura / SOC N1).

---

## Objetivo
Demonstrar habilidades práticas em:
- Atendimento e resolução de chamados
- Troubleshooting de redes e sistemas
- Automação com scripts
- Análise de logs (segurança básica)

---

## Tecnologias e Ferramentas
- Linux (Ubuntu)
- Windows / Windows Server
- Bash e PowerShell
- Redes (TCP/IP, DNS, DHCP)
- AWS (conceitos básicos)

---

## Estrutura do Projeto

- docs/ → Documentação técnica
- tickets/ → Simulação de chamados reais
- scripts/ → Automação e diagnósticos
- logs/ → Análise de segurança
- diagrams/ → Topologia de rede

---

## Cenários simulados
- Problemas de rede
- Reset de senha
- Máquina lenta
- Falha de DNS
- Ataques SSH (força bruta)
